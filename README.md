# 🧮 Simple Python Calculator

This project contains a simple interactive calculator script written in Python. 
It performs basic arithmetic operations with error handling and returns results rounded to **2 decimal places**.

## ✅ Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`)
- Input validation for:
  - Non-numeric inputs
  - Unsupported operations
  - Division by zero


##Technologies used
-Python3

## ▶️ How to Run

Make sure you have Python installed.

Open your terminal or command prompt and run: python calc.py
